# Teknoloji Yığını ve Gerekçeler

- **Frontend Framework:** React (Vite) - Bileşen tabanlı yapısıyla modülleri hızlıca geliştirmek için.
- **Stil ve CSS:** Tailwind CSS - Mobil uyumlu ve modern şefkatli tasarımı (mor/krem tonları) hızlıca kodlamak için.
- **Yapay Zeka (AI):** Anthropic Claude API - İnsan psikolojisini anlama, empati kurma ve kriz anlarında şefkatli bir persona (sistem promptu) yaratmak için endüstri standardı olduğu için.
- **Veri Saklama:** LocalStorage - MVP'yi hızla ayağa kaldırmak ve karmaşık veritabanı kurulumlarıyla vakit kaybetmemek için.
- **Yayınlama (Deploy):** Lovable / Netlify - Saniyeler içinde canlı link (URL) alabilmek için.
