# 🌸 Talya — PCOS'lu Kadınların Dijital Yoldaşı

> *"Diyet listesi değil — gerçek bir yoldaş."*

---

## 💜 Problem

Türkiye'de 200.000'den fazla kadın PCOS ile yaşıyor.

Ama mevcut sağlık uygulamaları bu kadınları görmüyor:
- Herkese aynı tarif, herkese aynı egzersiz — bütçe yok, mutfak yok, döngü evresi yok
- Yeme atağı geldiğinde, panik bastığında yanında kimse yok
- Beslenme bir yerde, egzersiz başka bir yerde, psikolojik destek hiçbir yerde

PCOS sadece fiziksel değil — insülin direnci, depresif ruh hali, yeme atakları hepsi bir arada geliyor. Hiçbir uygulama bütünü göremiyor.

**Talya görüyor.**

## 🤖 Talya Ne Yapıyor?

Talya, PCOS'lu kadınlara **üç alanda aynı anda** destek veren bir AI yoldaşıdır:

### 🥗 Kişiselleştirilmiş Beslenme
Bütçene, mutfak imkanlarına, alerjenlerine ve döngü evresine göre her gün farklı 4 tarif. Kısıtlı bütçeli öğrenci misin? Tek tencerede 15 dakika. Ovülasyon evresinde misin? Antioksidan zengin tarifler.

### 💪 Döngüye Uygun Egzersiz
Menstrüasyon, foliküler, ovülasyon ve luteal evreye özel yoga, güç ve düşük efor rutinleri. Vücuduna zorlamadan, döngünle birlikte hareket et.

### 🧠 Psikolojik Destek — Kriz Anında Yanında
Yeme atağı geldiğinde, panik bastığında — Güvenli Alan'ı aç. Rehberli nefes terapisi, Talya ile gerçek zamanlı chat ve hızlı olumlamalar. Çünkü bazen diyet listesi değil, sadece biri yanında olsun istiyorsun.

### 👭 Kız Kardeşlik Duvarı
Tamamen anonim topluluk desteği. PCOS'lu kadınlar birbirlerine mesaj bırakıyor: "Yalnız değilsin 💜"

## ⚙️ Kurulum

```bash
git clone https://github.com/semal13/Upschool.git
cd Upschool
npm install
```

`.env` dosyası oluştur:
```
VITE_GROQ_API_KEY=your_groq_api_key
VITE_N8N_WEBHOOK_URL=your_n8n_webhook_url
```

```bash
npm run dev
```
